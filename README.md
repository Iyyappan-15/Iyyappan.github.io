# Personal Portfolio Website

This is a responsive, static personal portfolio website built with HTML5 and CSS3.

## Project Structure
- `index.html`: Home page
- `about.html`: About Me page
- `skills.html`: Skills showcase
- `projects.html`: Projects gallery
- `contact.html`: Contact form
- `css/style.css`: Main stylesheet
- `assets/`: Images and icons directory
- `manifest.json`: PWA Manifest

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **PWA Ready**: Includes a `manifest.json` and `offline.html`.
- **Clean UI**: Modern design with blue and gray color scheme.
- **No Dependencies**: Pure HTML and CSS.

## How to Run
Simply open `index.html` in any modern web browser.

## Customization
- Edit the HTML files to update your content.
- Replace images in `assets/images/` with your own.
- Modify `css/style.css` to change colors or fonts.

## PWA Installation
To install as an app, open in a supported browser (like Chrome or Edge) and click the Install icon in the address bar.
